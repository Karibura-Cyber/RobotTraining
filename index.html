1-Day Training Syllabus: 2-Sensor Line Following Robot

Download Arduino 1.8.16 (INEX Edition) *optional*

Morning Session

Topic 1: Introduction to IPST-SE and Setup

  • Overview of IPST-SE board and components
  • Installing Arduino IDE and IPST-SE library
  • Basic program structure: setup() and loop()

Topic 2: Using the OK Button

The OK() function pauses execution until the OK button is pressed.

#include <ipst.h>

void setup() {
  OK(); // Wait until OK button pressed
  glcdClear();
  glcd(0,0,"Start Training");
}

void loop() {
}

Topic 3: Reading Sensor Values

Use analog(channel) to read sensor data (0–1023). For line tracking, connect left sensor to A0 and right sensor to A1.

#include <ipst.h>

int L, R;

void setup(){
  OK();
}

void loop(){
  L = analog(0); // Left sensor
  R = analog(1); // Right sensor
  glcdClear();
  glcd(0,0,"L=%d",L);
  glcd(1,0,"R=%d",R);
  delay(200);
}

Topic 4: Sensor Calibration & Thresholds

  • Find sensor values on white and black surfaces
  • Set a midpoint threshold for detection
int threshold = (450 + 50)/2; // Example calibration

if(L < threshold) { /* left on black */ }

Afternoon Session

Topic 5: Motor Control Basics

  • Moving forward: fd(speed)
  • Turning left/right: sl(speed), sr(speed)
  • Stopping: ao()

Topic 6: Basic 2-Sensor Line Tracking

#include <ipst.h>

int L, R;

void setup(){
  OK();
}

void loop(){
  L = analog(0);
  R = analog(1);

  if(L > 250 && R > 250){
    fd(80); // Both white → forward
  }
  else if(L < 250 && R > 250){
    sl(80); // Left black, right white → turn left
  }
  else if(L > 250 && R < 250){
    sr(80); // Right black, left white → turn right
  }
  else {
    fd(60); // Both black → junction → forward slowly
  }
}

Topic 7: Handling Junctions & Crossings

At intersections, the robot can beep or make a turn.

if(L < 250 && R < 250){
  sound(16,500,200);
  fd(60); delay(200);
}

Topic 8: Turning 90° with Functions

Create reusable functions for sharp turns at intersections.

void R90(){
  fd(60);
  sound(16,1500,100);
  while(analog(1) > 250){ sr(60); }
  while(analog(1) < 250){ sr(60); }
}

Topic 9: Adding Sound & Display Feedback

  • Use beep() or sound() to confirm sensor events
  • Display L and R values on GLCD for debugging

Topic 10: Speed Control & Smooth Turning

Adjust motor speeds to reduce wobble.

if(L < 250 && R > 250){
  motor(1,40); motor(2,80); // Smooth left turn
}

Topic 11: Stop at Goal Line

Robot stops completely when detecting a finish line (both black).

if(L < 250 && R < 250){
  ao();
  glcd(1,1,"Goal!");
  while(1); // Stop forever
}
